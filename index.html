Por que NotThereYet?

O nome vem de uma verdade simples: não importa onde eu esteja hoje, sempre haverá um próximo passo. Um idioma para aprender, uma habilidade para desenvolver, uma versão melhor de mim para construir.

Este blog é o registro desse processo — progresso acima de perfeição. Cada artigo nasce de experiências reais: frustrações, descobertas, recomeços e pequenas vitórias que, somadas, viram história.

Novos textos aparecem conforme a vida acontece. Volte quando quiser.
